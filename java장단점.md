 JAVA 의 장점은 독립성의 강하여서 어떤 프로그램에
적용이 가능하다는 점이 장점이고 객체지향 언어인 점과
라이브러리와 커뮤니티가 활성화 되어있어 배우기 좋고 접근성이 쉽다.

너무 많다.

되는데 java의 경우에는 import java.util.Scanner.....
